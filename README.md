# Daily Expense Tracker 💰

A simple, intuitive, and responsive web application to track and manage your daily expenses. Built with HTML, CSS (Bootstrap 5), and JavaScript (jQuery).

## Features

- **Full Dashboard Layout**: View your financial data at a glance.
- **Summary Cards**: Quick insights into your spending:
    - 📊 Total Expenses
    - 📅 Monthly Average
    - 🧾 Total Transactions
    - 💎 Highest Expense
- **Visual Analytics**: Interactive bar chart displaying your monthly spending.
- **Expense Management**: Add new expenses with Title, Amount (RM), and Date.
- **Yearly Filtering**: Easily filter your expenses and insights by year.
- **Responsive Design**: fully optimized for desktop, tablet, and mobile devices using Bootstrap 5.

## Technologies Used

- **HTML5** & **CSS3**
- **Bootstrap 5** (for responsive layout and styling)
- **JavaScript** (ES6+)
- **jQuery** (for DOM manipulation and event handling)

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/Shahrulzaman/Daily-Expense-Tracker.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Daily-Expense-Tracker
   ```
3. Open `index.html` in your favorite web browser.
   - Note: If you are using a local server (like XAMPP or VS Code Live Server), you can host it locally for the best experience.
